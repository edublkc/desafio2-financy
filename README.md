# Backend
- Após fazer o clone do projeto, faça uma copia do .env.example

- Instale os pacotes: `npm install`

- Faça as migrations executando: `npx prisma generate`

- Inicie o Servidor executando: `npm run dev`

# Frontend
- Após fazer o clone do projeto, faça uma copia do .env.example

- Instale os pacotes: `npm install`

- Faça as migrations executando: `npx prisma generate`

- Inicie o Servidor executando: `npm run dev`


### Observação

Se na hora de se cadastrar por algum motivo não funcionar.

Tente sincronizar o banco com o schema executando: `npx prisma db push` no backend.
